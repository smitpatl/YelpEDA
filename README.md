# Exploratory Data Analysis of  [Yelp Dataset](https://www.kaggle.com/yelp-dataset/yelp-dataset)

I performed some visualizations on the dataset to come up with business insights.

* [Matpoltlib](https://matplotlib.org/)
* [Seaborn](https://seaborn.pydata.org/)
* [Folium](https://python-visualization.github.io/folium/)
* [Plotly](https://plot.ly/python/)
